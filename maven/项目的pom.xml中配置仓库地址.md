## 配置阿里云镜像仓库

```
<repositories>
    <repository>
        <id>aliyun</id>
        <name>aliyun public</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </repository>
</repositories>
<pluginRepositories>
    <pluginRepository>
        <id>aliyun</id>
        <name>aliyun public</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </pluginRepository>
</pluginRepositories>
```
