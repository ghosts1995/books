
* `-version`    : 显示版本
* `-demuxers`    : 显示可用的 demuxers
* `-muxers`      : 显示可用的 muxers
* `-devices`     : 显示可用的输入输出设备
* `-codecs`      : 显示所有的编解码器
* `-decoders`    : 显示可用解码器
* `-encoders`    : 显示可用编码器
* `-bsfs`        : 显示比特流
* `-formats`     : 显示可用的格式
* `-protocols`   : 显示可用的协议
* `-filters`     : 显示可用的过滤器
* `-pix_fmts`    : 显示可用的像素格式
* `-sample_fmts` : 显示可用的采样格式
* `-layouts`     : 显示 channel 名称
* `-colors`      : 显示识别的颜色名称
