1. 查看使用的仓库：

```
$ yarn config get registry
https://registry.yarnpkg.com
```

2. 使用华为云镜像源

```
yarn config set registry https://repo.huaweicloud.com/repository/npm/
```

来源：https://mirrors.huaweicloud.com/
