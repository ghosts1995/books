## Alignment

## MainAxisAlignment

```
enum MainAxisAlignment {
    start,
    end,
    center,
    spaceBetween,
    spaceAround,
    spaceEvenly,
}
```

* start: 从左边开始
* end：从右边开始
* center：居中
* spaceBetween：各个元素之间间隔一样，但是前后两个元素挨着边缘
* spaceAround：各个元素之间间隔一样，但是前后两个元素不挨着边缘，距离边缘相同的距离
* spaceEvenly：各个元素之间以及距离两边边缘都一样
