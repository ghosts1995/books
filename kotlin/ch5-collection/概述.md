集合接口与相关函数位于 kotlin.collections 包中

---

List 的默认实现是 ArrayList，可以将其视为可调整大小的数组。

Set 的默认实现是 LinkedHashSet，它保留元素插入的顺序；另一种实现 HashSet 不声明元素的顺序。

Map 的默认实现是 LinkedHashMap，它保留元素插入的顺序；另一种实现 HashMap 不声明元素的顺序。
